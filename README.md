# Giorgio Saracco Website — Original Style Split Version

Questa versione mantiene il più possibile lo stile, il layout e i contenuti del file HTML originale.

## Cosa è cambiato

- Nessuna mappa
- Nessun cambio di concept grafico
- Nessuna riscrittura sostanziale dei contenuti
- Il file unico è stato separato in:
  - `index.html`
  - `styles.css`
  - `script.js`

## Come usarlo

1. Apri la cartella in VS Code
2. Modifica `index.html` se vuoi aggiornare testi, email o link
3. Modifica `styles.css` se vuoi ritoccare stile o layout
4. Modifica `script.js` se vuoi ritoccare interazioni/animazioni

## GitHub

Puoi caricare direttamente questi file nel tuo repo GitHub al posto del vecchio file unico.

## Nota

Questa è la versione giusta se vuoi preservare l'identità del sito originale e migliorare solo la manutenibilità tecnica.
